# Kolay Dil Ogren

Birlesik giris — **Tayca**, **Ingilizce** ve **Almanca** tam surumler.

## Uygulamalar

| Dil | Dosya | Dersler | Depolama |
|-----|-------|---------|----------|
| Tayca | `tayca-v3.html` | 48 | `tv3_*` |
| Ingilizce | `english-v3.html` | 49 | `en1_*` |
| Almanca | `german-v3.html` | 49 | `de1_*` |

Giris: `index.html` → arayuz dili + ogrenilecek dil → ilgili tam uygulama.

## Almanca paketi (v1.0.0)

- **20 senaryo dersi** + **29 bonus** kelime seti
- **Ders 2:** Artikel & sein (der/die/das, ein/eine) — detayli gramer
- Kelime orneklerinde **bd** (cumle parcalama), diyaloglarda **gramNote**
- Icerik: `languages/de/lessons/ders1.js` … `ders49.js`
- Uretim: `node tools/gen-german-lessons.mjs`
